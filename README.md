# Real-Time Financial Intelligence Platform

A production-style streaming financial intelligence system built with:

- Python
- Apache Kafka
- Real-time stream processing
- Parquet data lake
- Machine learning inference
- Quant analytics

## Architecture

Producer → Kafka → Stream Processor → Data Lake → ML Pipeline

## Features

- Real-time crypto ingestion
- Streaming feature engineering
- Online anomaly detection
- ML signal generation
- Parquet partitioned storage
- Quant backtesting framework

## Tech Stack

- Python
- Kafka
- Pandas
- NumPy
- Scikit-learn
- Docker
